# Note-Your-Task

**A simple web app where you can add tasks and make notes.**

This project is a lightweight, client-side task manager and note-taking application built with standard web technologies. It allows users to quickly jot down ideas, manage daily tasks, and keep track of important notes without the need for a complex backend or database setup.

## 🚀 Features

- **Task Management**: Add new tasks to your list.
- **Note Taking**: Create and view personal notes.
- **Simple Interface**: A clean and easy-to-use UI.
- **Client-Side Logic**: Runs entirely in the browser using JavaScript.
- **Responsive Design**: Styled with custom CSS for a pleasant user experience.

## 🛠️ Technologies Used

- **HTML5**: Structure of the application.
- **CSS3**: Styling and layout (45.1% of the codebase).
- **JavaScript**: Application logic and interactivity (27.7% of the codebase).

## 📂 Project Structure

The repository contains the following core files:

| File | Description |
